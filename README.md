## Getting Started

The easiest way to get started is to clone the repository:

```bash
# Get the latest snapshot
git clone https://github.com/simphiwehlabisa/ayoba-memestack.git

# Change directory
cd ayoba-memestack

# Install NPM dependencies
npm install

# Then simply start memestack app
node run start
```
